# Working folder del libro

Esta carpeta concentra la nueva base editorial en formato Markdown.

## Archivos

- `00-indice-propuesto.md`: estructura sugerida del libro.
- `01-diagnostico-editorial.md`: fortalezas, problemas y criterio de trabajo.
- `02-manuscrito-maestro.md`: primer manuscrito unificado.

## Criterio

- Los `.docx` originales se conservan como fuente.
- Las próximas iteraciones deberían ocurrir sobre Markdown.
- Cuando la estructura quede firme, se puede exportar otra vez a Word o Google Docs.
